## 运行环境

辅助的运行环境需满足如下全部条件.

 - 系统必须是**安卓**的, 且版本不高于安卓12

 - 分辨率必须为**9:16**, 不限横竖屏 (但推荐竖屏720x1280)

 - 必须已获取**Root权限**

对于Windows电脑, 推荐使用雷电模拟器, 它是虚拟的安卓设备, 自身拥有Root权限.

对于安卓手机玩家, 推荐使用红手指云手机、VMOS(请自行在百度或哔哩搜索免费版). 或者你希望对自己的安卓手机型号进行Root, 可查阅百度.

对于iOS苹果手机玩家, 需在安卓环境中配合辅助使用. 卡厄思梦境iOS和安卓账号互通, 可无缝切换.
